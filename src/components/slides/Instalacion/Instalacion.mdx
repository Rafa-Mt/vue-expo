# Como instalar Vue en su Proyecto
En esta sección aprenderás a crear una Aplicación de Página Única (SPA) de Vue en tu máquina local. El proyecto utilizará una configuración de compilación basada en Vite y te permitirá usar Componentes de Archivo Único (SFCs) de Vue.

### Requesitos previos
1. Asegurar de tener la version m,as actualizadad de Node.js
2. Abrir el centro de comandos del mismo directorio donde planeas hacer tu proyecto en vie.js

A lo que tengas estos requisitos, corre este comando:

```ps
npm create vue@latest
```

Este comando instalará y ejecutará **create-vue**, la herramienta oficial de Vue para crear la estructura básica de tu proyecto. Durante el proceso, se te preguntará acerca de características opcionales como TypeScript y soporte para pruebas, aqui se pueden ver:

```ps
✔ Project name: … <your-project-name>
✔ Add TypeScript? … No / Yes
✔ Add JSX Support? … No / Yes
✔ Add Vue Router for Single Page Application development? … No / Yes
✔ Add Pinia for state management? … No / Yes
✔ Add Vitest for Unit testing? … No / Yes
✔ Add an End-to-End Testing Solution? … No / Cypress / Playwright
✔ Add ESLint for code quality? … No / Yes
✔ Add Prettier for code formatting? … No / Yes

Scaffolding project in ./<your-project-name>...
Done.
```
Si no estás seguro de alguna opción, simplemente selecciona «No» presionando Enter. Una vez creado el proyecto, sigue las instrucciones para instalar las dependencias e iniciar el servidor de desarrollo.
